# UniInsight Dashboard 📈

A premium, AI-powered educational analytics platform that reimagines student performance tracking through the lens of a high-performance financial terminal.

![UniInsight Preview](https://raw.githubusercontent.com/username/repo/main/preview-image.png) *(Placeholder)*

## 🌟 The Philosophy
**UniInsight** treats academic progress as a valuable asset. We use the "Student-as-a-Stock" analogy to provide parents, teachers, and students with clear, actionable, and predictive insights into educational growth.

For a detailed breakdown of our mission, see [PROJECT_MOTIVE.md](./PROJECT_MOTIVE.md).

## ✨ Key Features
- **Visual Analytics**: Interactive performance charts inspired by TradingView.
- **AI Analytics**: Real-time signals and predictive risk indicators for early intervention.
- **Multi-Portal Access**: Dedicated dashboards for Students, Parents, Teachers, and Admins.
- **Placement Readiness**: Tracking competency levels against market demands.
- **Micro-Animations**: A smooth, reactive UI built with Framer Motion.

## 🚀 Tech Stack
- **Frontend**: React 19, Vite
- **Styling**: Vanilla CSS (Premium Dark Mode)
- **Charts**: Recharts
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Navigation**: React Router 7

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Project Structure
- `/src/pages`: Individual portal dashboards.
- `/src/components`: Reusable glassmorphic UI elements.
- `/src/context`: Authentication and global state.
- `/src/assets`: Design tokens and images.
